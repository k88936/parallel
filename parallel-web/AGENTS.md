# AGENTS.md - Development Guide for AI Coding Agents (parallel-web)

## Project Overview

This is the web frontend for the Parallel distributed AI coding agent management system. Built with:
- **React 19** with TypeScript
- **Vite** for build tooling
- **Redux Toolkit** for state management
- **Ring UI** (`@jetbrains/ring-ui-built`) for UI components
- **React Router** for routing

## Tech Stack

- React 19 + TypeScript
- Vite 7.x
- Redux Toolkit (with async thunks)
- Ring UI (JetBrains design system)
- Axios for API calls

## Commands

```bash
npm run build    # Build for production (runs tsc first)
npm run lint     # Run ESLint
```

## UI Component Guidelines

### Ring UI Components

Always prefer Ring UI components over custom implementations:

- **Dialog**: `import Dialog from '@jetbrains/ring-ui-built/components/dialog/dialog'`
- **Button**: `import Button from '@jetbrains/ring-ui-built/components/button/button'`
- **List**: `import List from '@jetbrains/ring-ui-built/components/list/list'`
- **Confirm**: `import Confirm from '@jetbrains/ring-ui-built/components/confirm/confirm'`
- **Island/IslandHeader/IslandContent**: For card-like containers
- **Tabs/Tab**: For tab navigation
- **Tag**: For labels/badges
- **Heading/Text**: For typography
- **Loader**: For loading states
- **Breadcrumbs**: For navigation breadcrumbs

### Form Styling

Use the shared `DialogForm.css` with ring-form CSS classes:

```tsx
<form className="ring-form">
  <span className="ring-form__title">Title</span>
  
  <div className="ring-form__group">
    <label className="ring-form__label">Label</label>
    <div className="ring-form__control">
      <input className="ring-input ring-input-size_m" />
      <div className="ring-error-bubble active">Error message</div>
    </div>
  </div>
  
  <div className="ring-form__footer">
    <Button primary>Submit</Button>
    <Button>Cancel</Button>
  </div>
</form>
```

### List Component Pattern

```tsx
<List
  data={items.map(item => ({
    rgItemType: Type.ITEM,
    key: item.id,
    label: item.name,
    description: item.description,
    rightNodes: (
      <>
        <Button onClick={(e) => { e.stopPropagation(); handleEdit(item); }}>Edit</Button>
        <Button danger onClick={(e) => { e.stopPropagation(); handleDelete(item); }}>Delete</Button>
      </>
    )
  }))}
  onSelect={() => {}}
  onMouseOut={() => {}}
  onScrollToBottom={() => {}}
  onResize={() => {}}
  restoreActiveIndex={false}
  activateSingleItem={false}
  activateFirstItem={false}
  shortcuts={false}
  renderOptimization={false}
  disableMoveDownOverflow={false}
  ariaLabel="Items"
/>
```

### Dialog Pattern

```tsx
<Dialog
  show={show}
  label="Dialog Title"
  onCloseAttempt={handleClose}
  onOverlayClick={handleClose}
  onEscPress={handleClose}
  showCloseButton
  trapFocus
  dense
>
  <div style={{width: 500}}>
    {/* Form content */}
  </div>
</Dialog>
```
