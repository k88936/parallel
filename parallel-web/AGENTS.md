# AGENTS.md - Development Guide for AI Coding Agents (parallel-web)

## Project Overview

This is the web frontend for the Parallel distributed AI coding agent management system. Built with:
- **React 19** with TypeScript
- **Vite** for build tooling
- **Ring UI** (`@jetbrains/ring-ui-built`) for UI components
- **React Router** for routing

## Tech Stack

- React 19 + TypeScript
- Vite 7.x
- tailwindcss
- Ring UI (JetBrains design system)
- Axios for API calls

## Commands

```bash
npm run build    # Build for production (runs tsc first)
npm run lint     # Run ESLint
```

## UI Component Guidelines

### Ring UI Components

Always prefer Ring UI components over custom implementations:

- **Dialog**: `import Dialog from '@jetbrains/ring-ui-built/components/dialog/dialog'`
- **Button**: `import Button from '@jetbrains/ring-ui-built/components/button/button'`
- **List**: `import List from '@jetbrains/ring-ui-built/components/list/list'`
- **Confirm**: `import Confirm from '@jetbrains/ring-ui-built/components/confirm/confirm'`
- **Island/IslandHeader/IslandContent**: For card-like containers
- **Tabs/Tab**: For tab navigation
- **Tag**: For labels/badges
- **TableContainer**: `import TableContainer from '@jetbrains/ring-ui-built/components/table/table'`
- **Pager**: `import Pager from '@jetbrains/ring-ui-built/components/pager/pager'`
- **Heading/Text**: For typography
- **Loader**: For loading states
- **Breadcrumbs**: For navigation breadcrumbs

For paginated data grids, prefer Ring UI's `TableContainer` + `Pager` instead of custom HTML tables. Keep row details/actions in a separate panel below the table when the expanded content is richer than a normal cell.

Example:

```tsx
import {useMemo, useState} from 'react';
import Pager from '@jetbrains/ring-ui-built/components/pager/pager';
import Selection from '@jetbrains/ring-ui-built/components/table/selection';
import TableContainer from '@jetbrains/ring-ui-built/components/table/table';
import type {Column, SortParams} from '@jetbrains/ring-ui-built/components/table/header-cell';

type CityRow = {
  city: string;
  country: string;
  id: number;
  url: string;
};

const data: CityRow[] = [
  {city: 'Sydney', country: 'Australia', id: 29, url: 'https://en.wikipedia.org/wiki/Australia'},
  {city: 'Brussels', country: 'Belgium', id: 14, url: 'http://statbel.fgov.be/fr/statistiques/chiffres/population/structure/natact/beletr/'},
  {city: 'Rio', country: 'Brazil', id: 9, url: 'https://en.wikipedia.org/wiki/Brazil'},
  {city: 'Sofia', country: 'Bulgaria', id: 21, url: 'https://en.wikipedia.org/wiki/Bulgaria'},
  {city: 'Beijing', country: 'China', id: 11, url: 'https://en.wikipedia.org/wiki/China'},
  {city: 'Helsinki', country: 'Finland', id: 6, url: 'https://en.wikipedia.org/wiki/Finland'},
  {city: 'Paris', country: 'France', id: 2, url: 'https://en.wikipedia.org/wiki/France'},
];

const columns: Column<CityRow>[] = [
  {id: 'country', sortable: true, title: 'Country'},
  {id: 'id', rightAlign: true, title: 'ID'},
  {id: 'city', sortable: true, title: 'City'},
  {
    id: 'url',
    title: 'URL',
    getValue: (item) => <a href={item.url}>{item.url}</a>,
  },
];

export function ExampleTable() {
  const [page, setPage] = useState(1);
  const [sortKey, setSortKey] = useState('country');
  const [sortOrder, setSortOrder] = useState(true);

  const sortedData = useMemo(() => {
    const nextData = [...data];
    nextData.sort((left, right) => {
      const comparison = String(left[sortKey as keyof CityRow]).localeCompare(String(right[sortKey as keyof CityRow]));
      return sortOrder ? comparison : -comparison;
    });
    return nextData;
  }, [sortKey, sortOrder]);

  const handleSort = ({column, order}: SortParams) => {
    setSortKey(column.id);
    setSortOrder(order);
  };

  const selection = useMemo(() => new Selection<CityRow>({
    data: sortedData,
    getKey: (item) => item.id,
  }), [sortedData]);

  return (
    <div>
      <TableContainer<CityRow>
        columns={columns}
        data={sortedData}
        getItemKey={(item) => item.id}
        onItemClick={(item) => console.log(item.id)}
        onSort={handleSort}
        selectable={false}
        selection={selection}
        sortKey={sortKey}
        sortOrder={sortOrder}
      />

      <Pager
        currentPage={page}
        disablePageSizeSelector
        onPageChange={(nextPage) => setPage(nextPage)}
        pageSize={7}
        total={30}
      />
    </div>
  );
}
```

## Types

Types in `src/types/` are auto-generated by `ts-rs` from Rust backend. Do not edit them manually.

binding gen from rust to ts:
```shell
# in project root:
make binding
```
